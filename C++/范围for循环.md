# 基于范围的for循环
在C++11中引入了新的for语句形式，语法如下：
```cpp
for (数据类型 迭代变量 : 范围)
{
//循环体
}
/*
语法中的数据类型也可以使用关键字auto，作用是让编译器自动推导出迭代变量的类型。和传统的for语句相比，基于范围的for循环不会出现数组下标越界的问题。
*/
```

**示例：**
```cpp
int arr[5]={10,20,30,40,50};
for(int i:arr)
{
	cout<<i<<"";
}
```

**还可以使用auto自动推导类型：**
```cpp
int arr[5]={10,20,30,40,50};
for(auto i:arr)
{
	cout<<i<<"";
}
```
